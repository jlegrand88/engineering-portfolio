**Goal:** Demonstrate modern fullstack architecture using React and Node.

### Stack
- Frontend: Next.js
- Backend: Node.js (Express)
- Auth: JWT
- CI/CD: GitHub Actions

### Architecture decisions
- API-first approach
- Separation of concerns
- Environment-based config
