**Objetivo:** Demostrar una arquitectura fullstack moderna utilizando React y Node.

### Stack
- Frontend: Next.js
- Backend: Node.js (Express)
- Autenticación: JWT
- CI/CD: GitHub Actions

### Decisiones de arquitectura
- Enfoque API-first
- Separación de responsabilidades
- Configuración basada en entornos

### Qué mejoraría
- Agregar pruebas automatizadas
- Migrar el backend a NestJS
