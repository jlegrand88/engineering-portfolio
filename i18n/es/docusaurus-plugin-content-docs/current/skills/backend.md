## Node.js

- Desarrollo de APIs REST

### Express.js
- Arquitectura basada en middlewares
- Validación de requests
- Flujos de autenticación (JWT)

### NestJS
- Arquitectura modular
- Controllers, services y providers
- Proyectos de aprendizaje y práctica

### Patrones Asíncronos
- Jobs en segundo plano
- Integración con APIs externas
- I/O no bloqueante y uso de async/await

### Configuración
- Configuración por entorno

## Laravel

- Arquitectura API-first

### Autenticación y Autorización
- Fortify, Sanctum
- Roles y permisos (Spatie)
- Policies y Gates

### Estructura de Aplicación
- Middlewares
- Capa de servicios
- Jobs y colas
- Tareas programadas (Cron)

### Patrones Orientados a Eventos
- Events y Listeners
- Procesamiento asíncrono
- Realtime y broadcasting (Ably, Reverb)

### Administración y Tooling
- Filament Admin

## Bases de datos

- Relacionales: MySQL, PostgreSQL, SQL Server
- NoSQL: MongoDB, Cloud Firestore (Google Cloud), DynamoDB (AWS)
- Embebidas / Mobile: SQLite (apps mobile offline-first)
