Esta sección aborda mi experiencia en desarrollo frontend, incluyendo frameworks modernos, estrategias SSR/SSG, manejo de estado, desarrollo móvil con Flutter y sistemas de interfaz accesibles y orientados al rendimiento.

## Aplicaciones Web

### React / Next.js
- Server-Side Rendering (SSR), Static Site Generation (SSG) y rendering híbrido
- Aplicaciones optimizadas para SEO
- Integración y consumo de APIs
- Arquitectura basada en componentes
- Manejo de estado (Hooks, Context)

### Vue / Nuxt
- Nuxt 3 con Composition API
- Server-Side Rendering (SSR) y Static Site Generation (SSG)
- Aplicaciones optimizadas para SEO
- Manejo de estado (Pinia)

### Integración con CMS Headless
- Strapi
- Arquitectura basada en APIs
- Modelado de contenido y control de roles

## Aplicaciones Móviles

### Flutter

- Desarrollo móvil Android

- Integración con APIs y capacidades del dispositivo
